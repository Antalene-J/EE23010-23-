1.1.4.The parametric equation for AB is given by

\begin{align}
$$\vec{x} &= \vec{A} + k\vec{m} $$\\
\text{where, }
$$\vec{m} &= \vec{B} -\vec{A}$$\\
\begin{split}
$$&= \myvec{-4 \\ 6} -\myvec{1\\ -1} $$\\
$$&= \myvec{-5 \\7} $$
\end{split}
\end{align}

Hence we get\\
\begin{align}
$$\text{AB: } \vec{x} = &\myvec{1\\-1} + k \myvec{-5\\7}$$
\end{align}


Similarly, 
\begin{align}
$$\text{BC: } \vec{x} = &\myvec{-4\\6} + k \myvec{1\\-11}$$\\
$$\text{CA: } \vec{x} = &\myvec{-3\\-5} + k \myvec{4\\4}$$
\end{align}
